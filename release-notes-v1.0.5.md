## 🎯 核心改动

### 1. 搜索匹配算法全面升级（爸爸要求）

旧的 score() 只看"标题包含/前缀"，新算法 `MatchScorer` 加入了 8 大改进：

1. **噪音词过滤**：剥离 HD / 1080p / 国语 / 粤语 / 蓝光 / BD / 高清 / TS / 4K / 双语 / 全集 / 完结 / 内嵌字幕 / 修复版 / 重制 等几十个常见标签
2. **多 token 命中**：搜"狂飙 张译"必须两个词都在标题里才高分；完全没命中会扣 2000 分
3. **连续字符奖励（LCS）**：连续命中 60 分/字符，离散命中 6 分/字符（差距 10 倍）
4. **完全相等 12000 分**、前缀 9000 分、包含 6000 分（按匹配度分档）
5. **年份接近度**：搜"三体 2023"，标题里"2023" +2000；其他年份按差距扣分
6. **拼音首字母兜底**：搜 "QZ" 时如果能拼出"狂飙"，加 600 分
7. **同源去重**：每个源最多保留 top 3，多出的每多 1 条扣 1500 分（防霸榜）
8. **二次搜索**：原关键词搜一遍，再把噪音词/括号剥掉搜一遍，合并结果

### 2. 豆瓣 API 内置（爸爸要求）

- **首页新增两行海报卡片**：
  - 豆瓣 · 热门电影（30 部）
  - 豆瓣 · 热门电视剧（30 部）
- **快捷入口新增"豆瓣热门"按钮**
- **豆瓣浏览页（独立 Activity）**：
  - 豆瓣首页（电影热门 + 电视剧热门 + 电影 Top250 + 电视剧 Top250 + 最新）
  - 豆瓣电影频道（热门/最新/Top250/经典/华语/欧美/日韩/动作/喜剧/爱情/科幻 等 16 个标签）
  - 豆瓣电视剧频道（17 个标签）
- **点击豆瓣卡片 → 聚合搜索真实视频源 → top1 直接进 DetailActivity 播放**
  - 搜不到时降级到浏览器打开豆瓣详情页
  - 搜索时自动生成多个 query（去括号内容/去年份/去季数），提高命中率
- **设置页新增豆瓣开关**（默认开启；可关闭）

### 3. 架构改进

- 新增 `MatchScorer` 工具类（搜索打分算法独立出来，豆瓣卡片和搜索页共用）
- 新增 `DoubanService`（独立服务，不实现 VideoClient 接口——豆瓣没播放源）
- 新增 `DoubanActivity` + `DoubanFragment` + `DoubanCardPresenter`
- 新增 `DoubanPosterPresenter`（首页用）
- 新增 `SettingsPrefs`（开关/偏好管理）

## ⚠️ 注意事项

- 豆瓣 API 来自 `https://movie.douban.com/j/search_subjects`，需要带 PC UA + Referer 才能拿到完整数据
- 部分豆瓣条目年份/标签可能不准确，搜索时如命中会跳到真实源播放
- 聚合搜索多源并发执行（每个源 1-2 秒），最多 5 个源约 2-3 秒完成
